# Vision Models

This repository contains experiments with modern computer vision architectures.

Currently implemented:

- **Swin-S Transformer**
- **MambaVision** (coming soon)

---

## Model 1: Swin-S Transformer

Dataset: HAM10000 skin lesion dataset  
Architecture: Swin-S Transformer  
Training Platform: Kaggle Notebook

### Files

- `swin-s-architecture.ipynb` — notebook describing the Swin-S architecture
- `ham10000-swin-s-training.ipynb` — includes exploratory data analysis (EDA), preprocessing, and training of the Swin-S model on the HAM10000 dataset.
